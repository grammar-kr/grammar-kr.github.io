all [ 통칭화 ]                  있는 대로 다 (이렇게 되어짐)         유튜브-영상 : shots

글자크기

영어 구문 안에서 어휘 all 에 대한 품사 구분은 어떻게 규정되어야 할 것인가.

영어 구문에서 all 의 쓰임들을 보면, 형용사 부사 대명사 등등의 품사적 기능으로
서로 다른 통사적 위치에 작용하고 있어서 구분하기가 쉽지 않다.




①  Take  all  of the apples. ( 그 사과 모두 가져라 )
②  Take it  all .  ( 그것을 모두 가져라 )



① 에서는 all 이 take 의 목적어인 대명사로 쓰였다고 볼 수 있겠지만
② 에서는 take 의 목적어가 대명사 it 이 되는데, 이어진 all 역시 대명사인가.

② 구문을 ① 구문의 또 다른 표현으로 볼 때

it 이 the apples 를 지칭한 것이면
all 은 그것의 수량 정도를 보충 서술한 형용사, 혹은 부사쯤으로 이해되어야 하는데

그렇다면 다음 ③ ④ 구문은 형용사인가 부사인가.




③  She is  all  kindness.                  ( 그녀는 매우 친절하다 )
④  She is  all  for your proposal. ( 그녀는 너의 제안에 전적으로 찬성한다 )



③ 의 우리말 '매우' 는 all 대한 해석이다.

kindness 가 명사형 어휘이고
all 이 그것을 수식하는 것으로 본다면 형용사로의 기능을 한 셈인데

여기의 all 에 '매우' 라는 우리말 적용은 형용사가 적절한 것인가.

④ 의 all 은
is 동사의 '보어' 로 기능하며 우리말 '전적으로' 로 해석되었는데

③ ④ 구문은
She is all 까지만 들으면 all 이 똑같은 의미로 인지될 수밖에 없다.

그렇다면
이어진 어구 때문에 all 의 품사적 기능과 의미가 달라지는 것으로 보아야 하는가

all 이 쓰인 구문들을 비교 분석하면서
심층적으로 작용하고 있다고 여겨지는 역할기능을 살펴보고

이 all 을 만족시키는 개연성 있는 개념을 소유해보자.
1. We do not all go.
( 우리는 모두 가는 것은 아니다 )

이 본문 1. 표현에서의 do 는 동사인가 조동사인가.
do 를 조동사로 보면, go 는 본동사(-원형동사)이어야 하는데

우리말 해석대로 분석하면 이 구문 형태는
all 을 조동사와 본동사 사이에 삽입한 부사쯤으로 간주해야하는 오류가 생긴다.

그럴지라도 분석을 위해
삽입된 부사로 이해해보면, 의미가 본문의 해석과는 조금 달리 나타나게 된다.

삽입 부사로 본 all 의 유, 무에서 비교해보자.




①  We do not        go . ( 우리는         가지 않는다 )
②  We do not  all  go . ( 우리는 모두 가지 않는다 )



all 이 없는 ① 구문은 '우리는 가지 않는다' 라는 의미의 부정형 구문이 되지만
go 앞에 all 을 삽입시켜 부사로 발화한 것이면

② 구문의 경우는 '우리는 모두 가지 않는다' 라는 표현이 되어
표현 의도가 '일부가 간다' 인 본문과는 다른 표현이 되고 만다.

다음에 제시된 ③ 과 ④ 같은
심층적 발화 메커니즘에 따른 구문 분석에서 보듯이

본문 1. 에서의 not 은 이어지는 어휘를 부정하므로
do 를 본동사로 이해해야만 한다.




③  We  do  +  not all  +  go


④ We  do   ourselves   +   ourselves   are done   +   it is   not all   +  it is   go
            ⓐ                                         ⓑ                      ⓒ            ⓓ
        ( 하게 한다 )                        ( 하게 되었다 )       ( 모두가 아니며), (그렇게 가는 것이다)


이와 같은 분석은, 심층 구조상으로
동사를 반드시 목적 대상에게 동작을 끼치게 하는 '사역성' 으로 타동사화 하고

주절 주어의 능동적 행위(ⓐ)에 따른 피역 대상의 수동(피동)적 처해짐(ⓑ)과
그것이 처한 상황에 대한 부가 서술(ⓒ)이

영어 구문 형성의 기저 현상임을 받아들이고 시도한 것으로써
이것은 다음과 같이 정리된다.


주어의 능동적 행위에 따른 의도대상의 수동(피동)화 현상이 영어구문 형성의 심층
기반에 있고 '주어 + 동사 + 대상(심층적 피동대상) + 상황구체화' 형식으로 나열된다.


이 분석에 따르면, We 가 자기 자신을 상대로 하는 심층사역 ⓐ 에 따라
그 자신은 그렇게 하게 된 수동적 상황 ⓑ 에 처해지며

그렇게 처한 상황 ⓑ 는 ⓒ 에서 it 에 의해 유추된 상황으로 지정되고
not all 의 부가로 일차적 구체화가 드러난다.

ⓒ 의 상황은 다시 ⓓ 에서처럼 go 라는 동작 형태가 부가되며
이차적인 구체화가 더해진 것이라고 보면

본문에서의 all 은 부사로 작용하는 것이 아니며
동사 go 의 의미상의 주어로 선택된 것도 아님을 알 수 있다.
2. We all have to go.
( 우리 모두는 가야 한다 )

본문 2.에서의 all 은
본문 1.과 비교할 때 쓰여진 통사적 위치가 다르다.

그렇다면 여기서의 all 은
삽입된 부사이거나 we 를 다시 강조하는 동격명사일 것이다.

영어에서의 동격명사 나열은 콤마(,)가 사용되는 것이 일반적인데
콤마의 사용이 없는 이 어휘나열에서 all 을 동격명사로 보면

앞의 어휘를 대신하는 대명사로 보아야만 하므로
부사라는 용어로 구분하는 것 자체가 필요 없게 된다.

이 구문에서 all 의 품사를 We 의 동격으로 보면 We = all 이 되는 것이므로
다음의 ① ② 경우처럼 각각 주어로 사용되면서 주격을 가질 수 있을 것이다.




①  We  have to go . ( 우리는 가야 한다 )
②  All  have to go . ( 모두는 가야 한다 )



all 을 동격으로 선택한 것이라고 보면
본문의 all 은 대명사로 작용해야 하므로 부사로의 기능이 없음은 분명하다.

왜냐하면 '부사' 라는 문법적 용어의 정의는
동사나 형용사, 다른 부사 혹은 문장 전체를 수식해주는 것으로 되어 있는데

이 구문에서는
의미상 We 을 대신하고 있어서 부사라고는 할 수 없다.

그렇다고 해서 이 all 을 발화한 화자의 선택 의도가
동격 명사의 삽입으로 어떤 의미 강조를 드러내려는 것으로 볼 수도 없다.

어휘 개념상  we =/= all  이라서
동격 명사가 아닌, 다른 의미가 드러나야 한다.
3. That's all.
( 그게 다야 )

that 은
'' 을 해주는 기능어로 구문에 작용한다.

이 본문 3. 에서도 that 은, 있었던 어떤 상황을 단정해 지칭한 주어이고
그 보어로 all 을 선택해서 그렇게 지칭된 어떤 상황의 보충서술이 되게 했다.

해석된 우리말 의미에 따르면
all 은 '있었던 것은 남김없이 다루어졌다' 는 것이므로

that 으로 지칭한 상황(대상)은 '이미 있던(있는) 것' 이 되며
all 에 의한 구체화 서술은 '남김없이 이러함' 이라고 할 수 있어서

all 을 우리말 '모두 다' '있는 대로 다' 라고 바꾸어도 그 의미가 전해진다.
다음의 ① ② 구문은 그 개념이 잘 적용됨을 보여준다.




①  All               is lost.       ( 만사가 글렀다 )
②  All  I want  is money. ( 내가 원하는 것은 돈 뿐이다 )



all 에 의해 지칭되는 것은 또다른 어떤 것이 남아있지 않다는 의미를 담고 있고
① 구문의 해석을 보더라도 우리말 '만사' 는 모든 것이다.

그런데, '만사' 라는 해석된 우리말 의미에서 보면

all 은 복수적 성향으로의 역할기능을 해보임이 분명한데도
① 구문에 사용된 동사를 보면 단수(형) is 이다.

또, 구문 ② 에서의 '돈 뿐' 이라는 해석 역시도
사용된 단수(형) is 때문에 돈 하나에만 국한시키는 단수적 의미가 되었겠지만

all 의 복수적 성향 그대로의 의미 지정을 수용하면
주어 I 가 원하는 것은 다수의 것이어야 한다.

다만 복수의 것들 하나 하나 모두가 '돈' 만을 원하는 것으로 표현되어 있어서
우리말 '뿐' 에 해당하는 의미로 '하나' 화 되어 묶였을 것이다.

그래서 두 가지 현상을 만족시키는 all 의 역할기능을
이런 일(것) 저런 일(것) 모두를 하나로 묶어 지칭해주는 '통칭화' 로 받아들이면

객체로의 대상들을 통합 상태의 대상으로 재지정 하는 것이 된다.




③  All  we can do  is   to wait for him .
( 그를 기다리는 수밖에 딴 도리가 없다 )


구문 ③ 의 의미에서도 all 의 단, 복수 성향을 인지할 수 있어야 하는데

주어가 할 수 있는 가능성(we can do)들은 all 안에 미지수로 설정되어 담기고
이어지는 동사에 의해서 그 대상의 수, 즉 가능성의 수가 규정되어 전해진다.

그리고 is 로는 '단수 상태' 만 드러낼 수 있으므로
to 전치사의 시도성, 즉 '' 를 반영하면

보어로 쓰인 to wait 는
all 에 담긴 '시도 가능성들' 가운데 '유일한 시도행위' 일 수 밖에 없기 때문에

이것을 all 의 '통칭적 성향' 으로 보고, 다음 구문에서 타당함을 확인해보자.
4. This is all yours.
( 이것은 모두 너의 것이다 )

4. 의 구문은 주어가 3인칭 단수라서 동사가 is 로 쓰인 구문이다.

이 표현에서 보어부의 all 을 빼면
This 로 지칭된 대상의 어떠함은 all 의 통칭화가 없게 되므로

yours 는 단수가 되어 본문의 표현의도와는 다른 구문이 된다.




①  This is yours . ( --단수화된 소유물-- )



이것은 본문 4. 에서의 all 이
yours 라는 복수의 것들을 묶어 하나처럼 '통칭화' 했다는 반증이다.

또 all 이
복수의 대상들 앞에서 직접 묶지 않았지만 is 동사의 보어로 성립됐다




②  All  of this is yours . ( 이 모두는 네 것이다 )



그리고 ② 에서는
this 때문에 is 가 선택된 것이 아니라, all(주어) 의 동사로 is 가 선택된 것이고

of this 는
all 의 보충어구이므로, 따라서 of this 를 빼도 구문은 성립된다.




③  All  is yours. ( 모두 네 것이다 )



① 에서처럼 보면
is 이하의 yours 가 단수화된 소유물로 인지되기 때문에

yours 자체로는 수의 어떠함을 알 수 없고
yours 가 단수인지 다수인지는 'all 의 통칭화' 유무에서 유추될 수 밖에 없다.




①  This   =   yours
④    All   =   yours




5. She lives all alone.
( 그녀는 오로지 혼자 산다 )

동사 live 는 '자동사' 이므로
심층에서의 동작사역이 주어 자신에게 행해집니다.

따라서 본문 5. 에서는 all 이 동사 live 의 목적어로 쓰인 것이 아니라
주어 자신의 수동적 처해짐의 '상태' 서술을 위해 쓰인 어휘이다.




①  She  lives   herself   +   herself   is lived   +  it is   all   +  it is   alone
                                                                                                                  ⓐ                 ⓑ


① 에서 ⓐ 의 it 은
is lived 라는 상태 지정 후 '그게 다' 임을 all 로 구체화 서술했고

다음 ⓑ 의 it 은 all 로 '통칭(-그게 모두이다-)된 구체화 상황' 을 다시 지정해
alone 으로 추가적 보충 서술을 한 것이다.

심층 개념적으로 해석해 보면
all alone 은 '통털어 혼자' 정도가 되는데

이 의미는 all 이 alone 으로 의미가 보충된 이후의 것일 때 비로소
'통칭화' 가 완성됨을 말해주는 것이다.

다시 말해서

all 의 통칭적 역할기능에 따른 작용개념으로 여겨지는 '있는 대로 다' 만으로는
is lived 의 수동 상황을 온전히 보충 서술해주지 못하므로

alone 으로
다시 부가 서술되어야만 온전한 통칭이 표현된다고 보는 것이다.

그렇다면 all 의 역할기능에 따른 작용개념에는
추가적 부가 서술에 해당하는 어떤 것이 작용하는 것으로 보이는데

다음과 같이 유추해 볼 수 있다.


② all = '있는 대로 다 (이렇게 되어 짐)'


이러한 개념은
다음 구문들에서 잘 적용되는 것으로부터 개연성을 찾을 수 있다.




③  She was dressed  all  in white . ( 그녀는 온통 하얀 옷을 입고 있었다 )
④  It's  all  one to me .                              ( 내게는 매 한가지다 )
⑤  It's  all  over .                                           ( 다 끝났다 )



③ 에서는 She 가 입게 된 여러 가지 옷들을 all 로 통칭하고 나서
'이렇게 되어짐' 을 in white 로 보충 부가한 표현이고

④ ⑤ 역시 같은 구조의 구문이다.
6. All the people I invited are coming.
( 내가 초대한 모든 사람이 오고 있다 )

영어 구문에서는
all + the + (대상 어구) 의 형태로 쓰인 구절을 많이 볼 수 있다.

우리말 그대로의 수식이 영어 수식에서도 발생한다면

all 이 'the + (대상 어구)' 를 '모든' 이라는 의미로
관형의 수식을 하고 있다고 볼 수 있겠지만

영어에서는 all 이
'the + (대상 어구)' 을 상대로 수식관계를 이루지 못한다.

the 는 이른바 '정관사()' 로써
대상을 ' 독자적인 그런 유형' 이 되게 하는 관형의 기능을 하기 때문에

the + (대상 어구) 은 독자적일 수밖에 없다.

다시 말해서 the 의 역할기능은
그만의 특정 대상이 되게 해서 대화에 선정해주는 것이므로

이것에 따르면
the 앞에 또 다른 수식의 관형이 성립되어서는 안 된다.

만약 그런 수식이 가능하면
그것은 이미 수식된 내용만큼의 또 다른 대상이 되는 것이므로

the 의 수식 의도에 따른
'특정화에 의한 독자성' 을 벗어난다고 할 수 있다.

따라서
표현상 수식이 필요하다면 the 아래로 들어와야 한다.



①  The man you met.
( 네가 만난 그 사람 )

②* Good the man you met.*( 네가 만난 착한 그 사람 ) -비문-
③  The good man you met .  ( 네가 만난 그 착한사람 )



본문 6. 에서처럼
the 앞에 쓰여 있는 all 은 the 를 넘어가지 못한다.

그래서 본문에서 all 을 빼고 살펴보자.



④  The people I invited  is coming .
 ( 내가 초대한 사람(들)이 오고 있다 )


구문 ④ 의 표현에서 초대에 응한 사람의 숫자는 얼마일까.

오고 있는 사람(들)이 주어 I 가 초대한 사람임은 알 수 있어도
그 숫자적 범위는 알 수 없다.

본문에서 '모두' 라는 의미는 all 에서 비롯된 것이며
어휘 all 의 사용은 초대한 사람들 전체 범위를 지정하는 것일 수밖에 없어서

지금까지 all 의 작용개념으로 도출한 바 있는
'있는 데로 다 (그렇게 되어 짐)' 을 적용해보면

초대한 사람들은 빠짐없이 다 오고 있다고 보는
통칭 선언이라고 할 만하다.

통칭 선언으로 수용을 하면
all 에 의해 통칭된 구성원은 '내가 초대한 사람들' 이므로

all 과 the people I invited 의 관계는
다음 ⑤ ⑥ 처럼 이해되어야 한다.


⑤ all 은 the people I invited 에 대한 '통칭화'
⑥ the people I invited 는 all 에 대한 '구체화'
7. All horses are animals, but not all animals are horses.
( 말들 모두는 동물이지만 동물들 모두가 말은 아니다 )

지금까지 살펴본 대로라면
all 은 뒤따라 이어지는 대상을 관형수식하지 않았다.

그렇다면 본문 7. 에서도 그랬어야만 하는데

우리말 해석과 우리 언어적 수식관계인 '말들 모두' 에 따르면
all 이 horses 를 관형수식을 한 것이라고 인지하게 된다.

all 이 horses 와 수식 결합되어 주어로 작용하고 있다면
이 결합에 의한 복합어는 주어로서의 격을 가지고 있을 것이므로

but 으로 종속되는 후속구절 역시도

all 이 animals 를 수식하면서 같은 주격으로 작용하여
다음 ① 과 같은 구문구조의 형태가 될 것이다.




① ( S  +  V  +  C ), but not ( S  +  V  +  C )

                        S = all + horses,
                        V = are,
                        C = animals, horses


그런데, 문법서에 따르면, but 으로 접속되는 구문에서
but 을 사이에 두고 나타나는 양쪽의 대립 범위는 같아야 한다고 하는데

① 의 분석 구조에서 보면
형태적으로 비대립의 어휘 나열 구문이 되고 만다.

예를 들자면, 앞 절에는 없으나 후속절에는 있는
not 의 존재를 설명하지 못한다.

그뿐만이 아니다. not 부정어가, 단순한 상황절, 즉

'하나의 상황으로 설정만해서 도입' 하는 '무지정(no-spec) 보문소 구문일 때
not 으로 부정하지 못하는 문법 현상을 수용하고 보면

not 의 존재를 해명하지 못한다.




② ...  not that the earth is round like a ball. (지구는 공처럼 둥근 것은 아니다)
     ...  not that (         S   +    V   +   C     )

③ ...  not(no-spec)the earth is round like a ball. (지구는 공처럼 둥그렇다가* 아니다)
     ...*not ( __ )    (         S   +    V   +   C     )


② 는 that 의 역할기능에 의해서
( S + V + C ) 로 표현되는 '' 한 것이고

그 상황절 앞에서 부정 기능을 하는 not 에 의해서는
그 '단정 사실' 이 부정된다.

그러나 ③ 에서는, (no-spec) 의 역할기능에 따르면

'지구가 둥굴다' 는 일반적 사실을 구문에 도입하기 위해서
하나의 상황으로 설정된 내용인데

그와 같은 일반적 사실을 부정하는 것 자체가
'' 표현하는 화자의 의도에서 보면 모순이 된다.

그런 부정 상황이 필요하다면
다음 ④ 구문에서처럼 미리 '부정의 상황으로 설정' 하는 것이 일반적이다.




④ ... (no-spec) The earth is not round like a ball. (지구는 공처럼 둥글지 않다)
     ... (no-spec)  (       S    +    V   not    C   )


따라서, 본문은 all 을 분리시켜
다음과 같은 구조로 심층에서 복원되어야 한다.




⑤      It is       all ( S + V + C ), but  it is not all ( S + V + C ).
(그것은 모두 이렇다 ( S + V + C ), 그러나       모두 그런 ( S + V + C ) 것은 아니다)


심층적 발화구조에서 보면
it 으로 유추되는 상황을 (not) all ( S + V + C ) 로 구체화 하는 형태가 되는데

이 상황 부가부에서의 구체화 요소인
all 과 ( S + V + C ) 에 작용하는 관계에서 생각해보면

all 은 ( S + V + C ) 모두를 '통칭화' 하는 보문소이고
( S + V + C ) 는 all 의 구성 요소이다.

이 같은 관계를 수용하고 본문을 보면
후속절의 not 이 all animals 를 부정한 것이 아니라

모든 동물이 다 그런 것이 아니라는 일반사실이 부정되었음을 알 수 있다.
8. He lost his all.
( 그는 그의 모든 것을 잃었다 )

이번에도 all 이 가지는 품사적 '격' 을 생각해보자.

지금까지 살펴본 바에 따르면

all 은 기능적으로는 '복수의 것(상황)을 통칭' 했고
통칭된 그것(상황)의 요소들이 있거나, 전제로 있었다.

그리고 all 의 역할은 주어에 대한 '보어' 이거나
하나의 구문 앞에서 그 문장의 성격을 선언하는 보문소처럼 작용했었다.

이 같은 현상을 받아들여서 일관되게 적용시키려면
본문 8. 에서의 all 또한 품사적 성향이 '보어' 로 작용하고 있음을 수용해야 한다.

우리말 해석 어휘대로라면
his 가 all 을 소유 수식한 것이 되어 명사처럼 인식하게도 만들지만

his 는 소유격만이 아니라 소유대명사로도 쓰이므로
다음과 같은 구문의 형태로 비교되어야 한다.




①*? I lost  all . ( 나의 모든 것 )?*
②  I lost mine  all . ( 내 것 모두 )


③*? You lost your  all . ( 너의 모든 것 ?* )
④  You lost yours  all . ( 네 것 모두 )



비교 제시된 ① 과 ③ 은
비문 여부를 떠나서 ② 와 ④ 의 구문 형태가 일반적이다.

이 구문들에서도 all 은
다음 ⑤ 의 구문 분석에서처럼 상황 부가부의 요소로 작용한다.



⑤ He lost    his   +   his   was lost   +  it was   all



화자는 주어 He 가 잃어버린 대상 범위를 '있는 대로 다' 라고 밝히고자 한 것이므로
all 은 his 가 처한 상태의 어떠함을 구체화 해주려는 보어이다.
9. All were happy.
( 다들 행복해 했다 )

본문 9. 에서의 all 은 우리말 해석에 따르면 복수로 표현되었는데
all 이 단수, 혹은 복수로 선택을 적용받을 때는 어떤 기준이 있을 것이다.

이것을 이해하기 위해서
다음 ① 과 같은 형태의 표현으로 바꾸어 도입해보자.




①  They were  all  happy . ( 그들은 모두 행복했다 )



① 의 표현에서 all 이 부사로 삽입되었다면, 동사가 were 인 구문이므로
주어에 대한 보어로서의 '격' 은 형용사 happy 에 부여되어야 한다.

그러나 분석해본 대로
all 이 '있는 대로 다' 라는 의미로 통칭해서 대상화되어 있음을 수용하면

주어의 대상(격)이 될 수 있어서
5형식의 '' 의 틀에서 분석해도 될 것이다.




②  They were  all  happy . ( S + V + O + C )


③  They were  all     +   all  were been   +  it was   happy .



심층적 구문구조로 나열해 본 ③ 의 상황부가부 요소인 it 은
앞의 심층적 수동 상태인 all were been 을 유추적으로 지정해서

다시 was happy 로 부가 서술한 것이므로 ④ 와 같은 관계가 성립된다.




④  all  were been   =  it   =  was   happy



all 의 상태가 happy 이므로
이 동등관계를 정리하면 all were(-was) happy 가 되며

이 심층 의미는
다음 ⑤ ⑥ 같은 두 가지 구문으로 표층화 될 수 있을 것이다.




⑤  All  were  happy . ( 모두 행복해 했다 )
⑥  All   was  happy . ( 모두 행복했다 )



본문 9. 에서의 all 은 우리말 해석에 따르면 복수로 표현되었는데
그렇다면 all 이 단수, 혹은 복수로 선택을 적용받을 때는 어떤 기준이 있을 것이다.

또 ⑤ 와 ⑥ 구문은
all 이 주어로 선택되면서 각기 다른 의미로 해석이 되었는데

이 통칭된 대상격 주어 all 에 대해서
was 와 were 가 선택되는 기준은 무엇이어야 하는가.

두 표층구문의 주어 all 은 행위격 주어가 아니라, 수동문의 주어처럼
대상격 주어로 작용했다고 보는 것이 해석이 주는 뉘앙스를 설명해 줄 수 있는데

⑤ 와 ⑥ 은 두 구문 모두에서 all 은 다수로 통칭했다.
그러나 all 에 대한 문법적 분석에 따르면

⑤ 에서는 were 를, ⑥ 은 was 를 선택한 것에 대한 해명 방법으로
all 을 이루는 대상이 무엇이냐에 따른 것으로 보기도 한다.

즉, all 이 사람을 다루는 경우이면 were 로 표현하고
동물을 다루는 경우 was 로 표현한다고 보는 듯하다.

사람은 각자를 독자적 인격체로 보기 때문이고,
동물은 한 무리로 취급되기 때문이라고 한다.

그러나
⑤ ⑥ 두 표현이 주는 의미에 따르면, all 은 대상격 주어로 작용했으므로

were(-was) happy 는
'있는 대로 다(이렇게 되어 짐)' 의 (이렇게 되어 짐) 에 해당한다.

그리고 두 구문은
were 와 was 에 의해서 all 의 단수 복수가 드러나게 되는데

were 는 유형-구성(are)의 당시형() 이라서 특성상 '서로가 있는 주어' 를 서술하고
was   는 유형-분간(is)  의 당시형()   이라서 특성상 '그만의 모습 주어' 를 서술하게 된다.

all 을 발화한 화자가 구성원 각각을 합으로 통칭했는지, 혹은
구성원 모두가 보여주는 상태를 하나로 통칭했는지에 따라 인지되어질 것이다.

역으로, 화자가 선택한 동사들의 다름만을 보고도
all 이 통칭하는 대상의 실체를 변별적으로 인식할 수 있게 되는 것도

all 에 '있는 대로 다 (이렇게 되어짐)' 이라는 개념의
'이렇게 되어짐' 이 작용하고 있다는 반증이 된다.


여기까지의 분석에 따르면
all 의 역할기능과 그에 따른 작용개념은 이렇게 된다.



첫째, all 은 삽입어가 아니라 '핵심어' 이다.

둘째, all 은 '있는 대로 다' 라는 범위를 정해 '통칭' 을 한다.

셋째, all 에 의한 '통칭' 은 그 구성 범위를 '이렇게 되어 짐' 으로 구체화한다.


역할기능 : 통칭화
작용개념 : 있는 대로 다 (이렇게 되어 짐)
