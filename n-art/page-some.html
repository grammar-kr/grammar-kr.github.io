some [ 나뉨-구별 ]        나뉘어 구분되어있는 '어떤' 것         유튜브-영상 : shots

글자크기

some 을 말 할 때
화자는 어떤 상태를 떠올리며 사용하는 것일까.

그리고 any 와는
나름의 의미적 대조를 이루는 듯한 문법 현상을 드러내기도 하는데

그런 현상을 어떻게 이해해야 하는 것일까.




①  Any  child can solve the problem .
  ( 어떤 애도 그 문제를 풀 수 있다 )
②  Some  child can solve the problem .
  ( 어떤 애는 그 문제를 풀 수 있다 )


수식어로서의 some 은
어떤, 일부, 다소, 등등의 우리말 의미로 번역되고 있지만

그 자체로는 명사처럼 혹은 다른 대명사와 결합되어
하나의 단어를 이루기도 하는데(something, someone, somebody,..)

이런 결합-대명사에서 작용하는
some 의 역할기능과 그에 따른 작용 개념은 무엇이어야 하는가.

이런 의문으로 some 을 살펴보아야 하는 것은

해석의 편의로 주어진 어떤, 일부의, 다소의, . . 등등의 우리식 사고로는
동의되어 지지 않는 것들을 경험할 수 있기 때문이다.




③  She put  some  water in a saucepan.
  ( 그녀는 소스 펜에 약간의 물을 넣었다 )
④  She put on  some  panties.
  ( 그녀는 팬티를 입었다 )


③ 에서의 some 은
우리 사전식 여러 개념으로도 잘 동의 되지만

그러나 ④ 에서의 some 은
우리 사전식 어떤 개념으로도 쉽사리 이해되지 않는다.

④ 의 경우는 '우리말' 화 할 때 번역을 하지 않는 것이 더 자연스러운데
그런 현상이 왜 생기는지를 some 이 가지고 있는 특성으로 이해해보자.
1. Some child can solve the problem.
( 어떤 아이는 그 문제를 풀 수 있다 )

본문 1. 에 쓰인 some 을 인지할 때
우리말 의미에 따르면, 구체적인 복수(-량)는 지칭하지 않았지만

약간, 조금, 일부, 어떤, 등등의
수량(-숫자)적 의미를 받아들여 번역하게 될 것이다.

그런데 이 구문에서 some 의 수식을 받은 child 는 단수로 쓰여 졌고
다음의 구문들에서 비교되듯이, 서로는 어떤 의미 차이도 낳고 있다.




①  Any  child can solve the problem .
  ( 어떤 애도 그 문제를 풀 수 있다 )
②  Some  child can solve the problem .
  ( 어떤 애는 그 문제를 풀 수 있다 )


문법서에서 some 과 any 만이 다른 두 구문을
'긍정과 부정' 으로 나누어 변별하기도 하지만

some 을 긍정문의 요소로
any 를 부정문, 의문문의 요소로 규정할 수 있을까.

먼저 ① 의 경우는
뒤에서 not 으로 동사를 부정해 보면 'any + not = No' 가 되어

③ 과 같이
'아무도 그 문제를 풀 수 없다' 라는 표현의 전체 부정문이 된다.




③  Any  child can not solve the problem.
  ( 어떤 애도 그 문제를 풀 수 없다 )


그렇지만 ① 구문은
부정어(-not)가 쓰여 있지 않기 때문에 any 가 쓰인 긍정문이다.

그렇다면 ① ② 모두 긍정문이 되는데
이들 서로 간에는 어떤 변별적 요소를 적용해야 하는가.

만약 ② 구문의 some child 에 적용되는 '어떤' 혹은 '일부' 라는 번역에
숫자(-수량)적 의미를 부여해야 한다면 ① 구문의 any 역시 그래야 할 것이다.

그런데
some 과 any 자체에서는 어떤 복수성을 찾아볼 수 없다.

왜냐하면 두 어휘가 뒤의(-수식하고자 하는) 명사에
'단수와 복수', '가산 명사와 비-가산 명사' 를 선택적으로 취할 수 있기 때문인데

다음 구문들은 그것을 보여주는 예이다.




④  He gave  some  money to  some  beggars.
  ( 그는 얼마의 돈을 몇 명의 거지들에게 주었다 )
⑤  He didn't give  any  money to  any  beggars .
  ( 그는 어떤 돈도 아무 거지들에게 주지 않았다 )




2. He asked for some pens, so I gave him some.
( 그가 펜을 요구해서 난 펜을 주었다 )

some 이 수식하고자 하는 대상을
단-복수 가산-비가산으로, 의도에 따라 선택적으로 취할 수 있다면

some 은
수량적 복수성으로 자질화된 수식어라고 볼 수는 없다.

본문 2. 의 앞 절과 뒷 절에 쓰인 some 은
수식 대상을 선택적으로 취하고 있는데

후속 구문에서의 some 이 수식어가 아니라, 대명사로 쓰였다 해도
앞에서 언급된 some pens 의 s 까지 대신한다고 할 수는 없다.

만약 s 까지를 대신하는 수량적 대명사라면
so 이하의 표현에서 화자 I 가 여러 자루의 펜을 주었다고 인지해야 하는데

해석이 주는 의미에 따르면
여기서의 some 에서는 수량(숫자)적 의미는 드러나지 않는다.

실제로 him 이 여러 자루의 펜을 받았다면 당연히 복수가 주어져야 하고
그럴 때라면 some ones 로 표현되었을 것이다.

후속절에서의 주어 I 가 him 에게 준 것이 여러 자루의 펜이 아니라
그것 중 '일부' 를 말하는 것임을 다음 구문들에서 확인해 보자.




①  Do you want  any  of these books?
  ( 너는 이 책들 중에서 어떤 것이라도 원하니? )
②  Yes I want  some .
  ( 응 어떤 책이 필요해 )


① 과 ② 대화에서 보면 질문은 any 로 했고, 대답은 some 으로 했다.
여기서 대답의 3가지 가능성을 놓고 생각해보자.




③  Yes, I want          some  of them . ( 응 어떤 책을 원해)
④  No,  I don't want  any  of them . ( 아니, 어떤 책도 원치 않아 )
⑤  Yes, I want   any  of them . ( 응 아무 책이라도 좋아 )



③ ④ ⑤ 모두
some, any 의 전체 구성 대상으로 them 을 지정한 것이라서

이들이 지칭해주는 대상은 them 의 일부분일 수밖에 없을 것이므로
어휘 서로는 다음과 같은 의미 차이를 낳는다.


③ 의 경우 '어떤 일정한 책을 원한다'
④ 의 경우 '아무 책도 원치 않는다'
⑤ 의 경우 '어떤 책이라도 가리지 않고 원한다'


③ 구문 표현에서 some 은 them 에 속한 어떤 책을 지칭하고 있지만
몇 권인지는 드러내지 않는다.

다만 그것이 them 의 '일부분' 라는 것만은 분명하다

이것으로 부터
화자가 some 을 선택할 때 가산적 수량을 말하려는 것이 아니라

비가산적 나뉨량을 표현하기 위해서 선택하는 어휘로 보고
이제는 '나뉨' 을 전제로 생각해보자.
3. If anybody wanted to tell me something, they'd have to write on paper.
( 누군가 내게 할 말이 있다면 적어 보내라고 해 )

본문 3. 은 me 에게 어떤 말을 전하고자 한다면
그 사람이 누구든 글로 적어 보내야 할 것이라는 표현인데

이 구문 표현에서
화자의 발화 의도 대상은 '누구든지' 이다.

any 와 some 이 함께 쓰여 있는 앞 조건절은
anybody 가 주어로 발화되어 있는데

누구라고 특정인을 지칭하지 않는, 그렇지만
모두가 특정인일 수 있는 '누구든지' 를 드러내는 어휘는 any 이다.

우리말 해석어에 따르면
anybody 가 me 에게 어떤 말을 전하고자 했을 때는

주어 anybody 의 마음속에 이미
'어떤 것' 이 정해져 있다고 화자 I(-me)는 단정을 하고 이 말을 한 것이 된다.

따라서 화자는 그 단정의 의미가 들어있는 some 을
의도적으로 선택해서 발화한 것이 분명해진다.

그리고 이것은 some 이
'특정 대상을 결정지어주는 요소' 를 가지고 있다고 생각해 볼 수 있게 한다.

그런데
그 특정 대상이 결정될 때는 '나뉨' 이 전제되어 있을 것이다.

다시 말해서 전하고 싶은 말은, 많은 말 중의 '어떤 것' 으로,
something of everything 에 해당한다.

이렇듯이 everything 으로부터 나누인 일부가 something 이므로
수식어 some 은 대상을 나누어 구별시켜주는 역할을 하고 있다고 보아야 한다.




①  It was  some  party.
  ( 대단한 연회였다 )


① 의 해석어 중에
'대단한 파티였다' 에서의 '대단한' 에 해당하는 것은 some 이다.

이런 의역이 가능한 것도
some 뒤에 '보통 행해지는 모든 파티' 정도를 심층 배경으로 두고

그것으로부터
'구별' 되는 파티' 라는 내포된 의미가 있기에 가능하다고 여겨진다.
4. There is some truth in what you say.
( 자네 말에도 일리가 있네 )

본문 4. 에서 some 의 수식을 받는 truth 가
셀 수 없는 추상 명사이기 때문이기도 하지만

이 표현에서의 쓰임을 보더라도
some 은 결코 '계수' 적 수식을 하고 있지는 않다.

you 가 한 말을
'수긍이 가는' truth 라고 수식해주는 의미 지정이 드러날 뿐이다.

그래서 some 이 주로 긍정문에서
any 가 주로 긍정문이 아닌데서 많이 쓰이는 것을 근거로 해서

다음과 같이 규정하고 서로를 상반되게 대조할 수 있다.




①   some   =   단정
②       any    =   비단정



본문 4. 에서 some truth 를 '일리' 라고 해석했는데

some truth 를
화자의 생각 속에서 '어떤 것이다' 라고 구체적 지정은 하지 않았지만

그 존재는 인정해주는 '단정' 이 작용하기 때문에
그렇게 인지되어 질 것이다.

만약 any 로 말했을 경우라면 '일리' 즉 any truth 는
의미상 '어떤 것이다' 라는 구체적 지정을 못하기 때문에

그 존재가 인정될 수 없는 '비단정' 을 적용받아서
some 과는 상반된 의미를 드러내게 되고

부정어 not 을 수반하고서
부정적인 의사 표현에 선택되어지는 것으로 보인다.




③  There is      some  truth in what you say . ( 일부분 일리가 있음 )
④  There is not  any  truth in what you say . ( 전혀 일리가 없음 )




5. May I have some more milk?
( 우유 좀 더 마셔도 될까요? )

여기서는 그 같은 단정과 비-단정을 적용해보자.
이 본문은 의문문인데도 any 를 사용하지 않았다.

그것은 살펴본 바대로라면
any 는 대상의 존재를 구체적으로 지칭하지 않기 때문이어야 한다.

화자가 마시고자 한 것은 우유이고
대화 내에서 뚜렷하게 인식되는 존재다.

milk 를 '더' 마시고 싶다고 말한 것은 이미 milk 를 마셨다는 것이므로
다른 음식과는 분명히 구별되는 음료로 지칭한 것이 된다.

따라서 some 의 단정적 기능이
milk 를 구별된 음료로 수식 지칭해서 구체적인 대상이 되게 했다고 할 수 있다.

다음 ① 은
화자가 상대에게 권하는 것이 커피이고, 구체적 지칭을 한 표현이다.




①  Will you have  some  coffee?
  ( 커피 좀 마실래? )


① 의 표현에서도 화자가 선택해서 권하는 음료는
coffee 라는, 다른 것과는 구별된 음료임을 알 수 있다.

이 표현에서 milk 나 coffee 대신에
drink 와 같은 표현의 범위가 포괄적인 어휘가 왔다면

'어떤 것' 이라고 지정하지 못하는 비-단정적 선택의 질문을 던진 꼴이 되어
any 가 사용되어도 자연스럽게 들릴 것이다.




②  May   I    have  any  more drink? ( 음료수 더 마셔도 될까요? )
③  Will you have   any  drink? ( 뭐라도 좀 마실래? )




6. If there is any more of this stuff, I'll get some
( 이 재료가 좀 더 있다면 얻어 가겠는데 )

본문6. 은 앞 선행절이 '조건문' 이다.

그리고 if 는
'' 를 구문에 선언해서 상황을 가정해주는 기능을 한다.

이 본문이 표현하는 의미에서 생각해 본다면

상황을 가정한 선행절에서는 if 의 역할기능대로
'혹 그럴 가능성의 경우' 를 단정 지어서 말하지는 못할 것이므로

some 으로 대상을 단정적으로 수식할 수 없을 것이다.

따라서 any 로의 비단정(-비구별) 수식은
필요로 하는 그 재료를 나누어 구별하지 않는 '어떤 것이라도' 라는 표현이 되게 한다.

그러나 이어지는 주절에서는
앞에서 그 존재를 이미 가정해 둔 이후의 발화이기 때문에

일반적 stuff 가 아니라, 가정적으로 존재시켜 둔 그 stuff 를
some 으로 구별되게 해 준 것이라 할 수 있다.

즉, 재료가 어떤(-any) 것이든
거기에 존재하는 그것(-some)을 가져가겠다는 것이다.
7. We want someone of you to carry this to the station.
( 우리는 너희 중 누가 이걸 역으로 날라다 주길 원해 )

본문 7. 은 some 과 one 이 결합했을 때도
똑같은 수식적 요소가 작용하고 있는 구문이다.

someone of you 는 '너희 중에 누군가가' 라는 의미인데
some 이 you 중의 하나인 one 을 구체적으로 지칭하지 않았어도

누군가 한사람이 해주기를 바라는
단정적 선언으로 수식을 해서 someone 으로 결속시킨 셈이다.

반면에 some 의 위치에 any 가 쓰였다면
그 대상은 구체적이지 못함을 볼 수 있다.




①  We want  anyone of you to carry this the station.
  ( 너희 중에 아무라도... )
②  The moderns must know  something of everything, and everything of something.
  ( 현대인들은 상식과 전문지식이 필요하다 )


you 중에서 '아무라도' 해달라는
선택에 있어서의 비단정적(-비구별) 의사 표현을 하고 있는데

② 의 표현에서 보면
some 과 thing 의 결합에서도 같은 수식이 작용한다.

다음의 ③ 과 ④ 는
something 과 everything 의 혼합구문으로써

and 를 두고 something 과 everything 이
핵어와 보충어 관계를 바꾸고 있다.




③  something  of  everything . ( 모든 것 중에서 구별되어 진 어떤 것 )
④  everything  of  something . ( 구별되어진 그 어떤 모든 것 )



③ 에서는
'모든 것(-everything-) 중에서 구별되어 진 어떤(-something-) 것' 를 말하고

④ 에서는
'구별되어진 어떤(-something-) 것들을 총칭한 모든(-everything-) 것' 이다.

사람이 모든 것을 다 알고 살지는 않는다.
그것들 중 어떤 것들은 알고 살아가게 되는데

② 의 and 앞은
그것(-알고 살게 되는 것)이 모든 것에서 구별되어지는 '어떤 것' 에 해당하고

'상식' 이라는 의역을 낳은 셈이다.

그리고 ② 의 and 이하 구절은
'어떤 것으로 구별되어진 그런 것들 전부' 를 말하는 것이므로

'전문지식' 이라는 의역이 타당해진다.
8. She puts on some panties.
( 그녀는 바지를 입는다 )

본문 8. 의 some 은
우리말로 번역할 때 적절한 어휘 적용이 쉽지 않을 것이다.




①  Jane puts on   some  underwear. ( 제인은 속옷을 입는다 )
②  Jane puts on   some  socks. ( 제인은 양말을 신는다 )
③  Jane puts on   some  pants. ( 제인은 바지를 입는다 )
④  Jane puts on   some  glasses. ( 제인은 안경을 쓴다 )


⑤  Jane puts on   a  skirt. ( 제인은 치마를 입는다 )
⑥  Jane puts on   a  dress. ( 제인은 드레스를 입는다 )
⑦  Jane puts on   a  shirt. ( 제인은 셔츠를 입는다 )



제시된 표현들에서
some 이 수식하는 대상들의 공통점을 보면 모두가 복수형이다.

그렇지만
단수만으로는 제 기능을 하지 못하는 단일-복수형들임을 알 수 있는데

나머지 a 의 수식으로 표현되는 대상들을 보면
모두가 단수형이고, 그 단수로도 충분히 제 기능을 하는 것들이다.

따라서 some 은 a 와 변별되는
그만의 의미를 소유하고 있다는 것을 보여주는 셈이다.

a 가 단수형 수식어이기 때문에
some 이 복수일 것이라는 분석은 재고되어야 한다.

왜냐하면, a 의 단수 지칭이 의도적 가산이 아니듯
some 역시 결과적으로 복수를 내포할 뿐이기 때문이다.

그리고 제시된 위의 구문들에서 보면

some 의 수식을 받는 대상들의 공통점은
좌우 대칭적으로 나뉘어 구별되어 있음을 알 수 있는데

⑥ 구문에서와 ⑦ 의 구문에서는
상의인 a blouse, a shirt 등도 좌우 대칭이 있는데도 a 의 수식 대상이 되었다.

그래서
반드시 있어야 하는 개념적 요소인 '나뉘어' 를 적용시켜야 한다.

상의나 치마는 하나이거나 하나로 간주될 뿐, 나뉘는 대상은 결코 아니다.
또 some 의 수식어 때문에 socks 가 복수로 표현되는 것도 아니라는 점이다.

① ② ③ ④ 구문은 그 대상들이 '단일-복수형' 이기 때문에
a 가 아닌 some 으로 수식을 하고 있는 구문들인데

우리말로 번역을 할때, 적절한 말을 찾기 어려운 것을 보면

some 은 자신을 포함해 수식하는 대상을
'나뉘어 구별되어진 것' 으로 지정해주는 역할을 하고 있다고 할 만하다.


지금까지 some 이 쓰인 구문을 정리하면 이렇다.



첫째, some 은 수량에 대한 한정이 아니라 '구분되어 있는 것의 지정' 이다.

둘째, some 은 긍정의 요소가 아니라 '단정' 의 요소이다.

셋째, some 은 심층에 전제된, 전체로부터 나누어진 '어떤' 것이다.
