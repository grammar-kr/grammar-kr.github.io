any [ 나뉨-비구별 ]        나뉘어 구분되어지는 것 '전부'         유튜브-영상 : shots

글자크기

any 를 발화할 때
화자는 어떤 상태를 떠올리며 사용하는 것일까.

그리고
some 과는 그 나름의 의미적 대조를 이루는 듯한 문법 현상을 보여 주는데

그런 현상을 어떻게 이해해야 하는 것일까.




①  Any  child can solve the problem .
( 어떤 애도 그 문제를 풀 수 있다 )
②  Some  child can solve the problem .
( 어떤 애는 그 문제를 풀 수 있다 )



그리고 any 는
아무도, 어느 것도, 어떤 것도, 등등으로 번역되고

그 자체로 명사처럼 혹은
다른 대명사와 결합(-anything, anyone, anybody)해서 하나의 단어가 되는데

any 가 결합된 대명사와 수식 상태의 any 는
어떠한 의미로 차이를 보이는 것일까.




③  Any  one of you will do . ( 아무나 한사람 해주면 돼 )
④  Anyone   of you will do . ( 누구든지 간에 하려든다 )



또 any 는
우리말로 번역을 해봐도 대부분 별다른 어려움이 느껴지지 않는다.

그러나 any 를
의문문, 조건문, 부정문 등을 이루는 것으로 규정한 문법 분석에 따르다 보면

다음의 긍정문 해석에서는 망설이게 될 것이다.




⑤  He has  any  number of books .
( 그는 많은 책을 가지고 있다 )


이 구문에서 any 에 대한 우리말 해석을 보면
any 가 '많은' 이라는 무특정 다수를 지칭하는 수식어로 번역이 되는데

왜 이렇게 되는지를
any 의 특성으로 살펴보며 그 역할기능과 그에 따른 작용개념을 도출해보자.
1. Any child can solve the problem.
( 어떤 아이도 그 문제를 풀 수 있다 )

any 가 쓰이는 구문 대부분에서, any 를 번역한 우리말 의미를 보면

구체적인 수량은 드러내지 못하지만, 아무도, 어느 것도, 어떤 것도, 등등으로
복수적(-다수적) 의미가 잠재되는 번역을 하는 것을 볼 수 있는데

이 구문에서 any 의 수식을 당하는 대상 child 는 단수이다.
그렇다면 any 를 단수형 수식어로 보아야 하는 것일까.

또 다음의 ① ② 구문을 보면 서로가 어떤 의미 차이도 낳고 있다.




①  Any  child can solve the problem .
( 어떤 애도 그 문제를 풀 수 있다 )
②  Some  child can solve the problem .
( 어떤 애는 그 문제를 풀 수 있다 )


문법서에 따르면
some 과 any 만이 다른 두 구문을 '긍정과 부정' 으로 나누어 변별하기도 하지만

some 을 긍정문의 요소로
any 를 부정문, 의문문의 요소로 규정할 수 있을까.

① 의 경우는 뒤에서 not 으로 동사를 부정해보면 any + not = No 가 되어
③ 에서처럼 '아무도 그 문제를 풀 수 없다' 라는 표현의 전체부정문이 된다.




③  Any  child can not solve the problem .
  ( 어떤 애도 그 문제를 풀 수 없다 )


그렇지만 ① 은
not 이 쓰여 있지 않기 때문에 any 가 쓰였지만 긍정 구문이다.

그렇다면 ① ② 모두는 긍정문이 되는데
이들 서로 간에는 어떤 변별적 요소를 적용해야 하는가.

만약 ① 의 any child 에 적용되는 '어떤' 혹은 '일부' 라는 번역에
숫자(-수량)적 의미 부여를 하면

② 의 some 역시 그래야 하기 때문에
any, some 자체에서는 어떤 복수성을 찾아서는 안 될 듯하다.

왜냐하면 두 어휘가 뒤의(-수식하고자 하는) 명사에 단수와 복수 그리고
가산명사와 비가산명사를 선택적으로 취하는 것을 볼 수 있기 때문인데

다음 두 구문은 그 예가 된다.




④  He gave  some money to some  beggars .
  ( 그는 얼마의 돈을 몇 명의 거지들에게 주었다 )
⑤  He didn't give  any money to any  beggars .
  ( 그는 어떤 돈도 아무 거지들에게 주지 않았다 )




2. Have you any eggs to boil?
( 삶을 계란 좀 있니? )

앞에서의 구문들과는 달리
any 로 수식받는 명사 eggs 가 복수로 표현되어 있다.

만약 any 가 자체적으로 복수성을 지니고 의미 지정을 한다면
egg 단수(-속성 명사)만으로 충분할 것이다.

앞에서 예시한 구문에서 보았듯이 any 는 단, 복수 모두를 취하는 것으로 보아
그 자체로 수량적 지칭을 하지는 않는 듯하다.

다음의 ① 구문처럼 수량을 생각할 수 없는 표현에서도 사용되기 때문이다.




①  You may visit me  any day .
  ( 아무 날에나 방문해도 좋다 )


아무 날이라고 한 의미에서 보면

day 는 계수하는 날이 아니라, 정해지지 않은 특정일이라서
day 자체는 속성명사로 선택된 것이다.

애써 수량적 의미 부여를 한다면
그것은 단수이고, 그것도 총칭에 따른 단수다.




②  Do you want  any  of these books ?
( 너는 이 책들 중에서 어떤 것이라도 원하니? )

③  I want some.
( 응 어떤 책이 필요해 )



② 와 ③ 의 대화에서 보면 질문은 any 로 했고, 대답은 some 으로 했다.
여기에서 대답의 3가지 가능성을 놓고 생각해 보자.




④  Yes, I want          some  of them . ( 응 어떤 책을 원해)
⑤  No,  I don't want  any  of them . ( 아니, 어떤 책도 원치 않아 )
⑥  Yes, I want   any  of them . ( 응 아무 책이라도 좋아 )



④ ⑤ ⑥ 모두
any, some 의 전체 구성 대상으로 them 을 지정한 것이라서

이들이 지칭하는 대상은 them 의 일부분일 수밖에 없을 것이므로
어휘 서로는 다음과 같은 의미 차이를 낳는다.


④ 의 경우 '어떤 일정한 책을 원한다'
⑤ 의 경우 '아무 책도 원치 않는다'
⑥ 의 경우 '어떤 책이라도 가리지 않고 원한다'


④ 의 some 은
them 에 속한 어떤 책을 지칭하고 있지만 몇 권인지는 드러내지 않는다.

them 의 '일부분' 에 속한 것을 전제로 하는 것으로 인지되게 할 뿐이고
⑤ 에서의 경우에도 them 의 일부분이어야 하는데

우리말 '어떤 책도' 에 따르면

them 에 속한 일부분(-them 전체일 수도 있음-)을 이루는 요소들을
총칭해서 전제로 하는 불특정 하나 하나 모두를 지칭한 것이 된다.

이에 따르면, any 는 가산적 수량을 말하는 어휘가 아니라
'비가산적 나뉨량' 을 표현하는 어휘로 보아야 하므로

이제는 '나뉨' 을 전제로 생각해보자.
3. He is younger than any of his school-fellows.
( 그는 어떤 학우보다도 어리다 )

이 구문은
'그 학우들 중 누구보다도 젊다' 라는 의미를 전하고 있다.

즉, 학우들 모두는
'이사람 저사람 가릴 것 없이' 모두 he 보다는 늙었다는 말이다.

이 구문 전체가 드러내는 의미에 따라
다음과 같은 현상을 가정하고 생각해 보자.

any + not = No 가 되어
'전체부정' 을 하는 표현형이 된다고 보는 문법에서 볼 때

본문 3. 은 not 이 없으므로 any + (    ) = yes 가 되는 셈이고
따라서 '전체긍정' 을 드러내는 표현형이 된다.



① any + not = '전체부정'
② any + (     ) = '전체긍정'



이 현상 그대로를 some 으로 대치해 보면

나뉘어 구분되어 있는 '어떤' 것을 지정하는 some 의 구문에서는
다음 ③ ④ 처럼 생각해 볼 수 있다.



③ some + not = '부분부정'
④ some + (     ) = '부분긍정'



이런 상반된 현상에서 보면

some 의 대상 지칭은 '부분' 을 대상으로 하는 것이고
any 는 '전체' 를 대상으로 하는 것이 된다.

그러나, any 가 '전체' 를 대상화할 때는

전체를 하나로 취급하는 것이 아니라
전체를 이루는 구성자 하나 하나를 대상으로 한다.




⑤  He would do  any job for money .
  ( 그는 돈이라면 무슨 일이라도 할 것이다 )


모든 직업에 대해 '가리지 않고 하겠다' 는 의미에서도 그렇고
'나뉨' 을 적용해보아도

any 로의 대상 지칭은, 전체가 아니라
전체를 이루는 '하나하나' 가 총칭된 것임을 유추할 수 있다.
4. I should like to know  whether you have any trouble at home.
( 난 너희 집안에 무슨 문제가 있는지 알아야 돼 )

whether 로 이끌어지는 목적절이 조건문이라서 any 가 쓰였다고 이해한다면
표층적 현상만을 두고 분석하는 꼴이다.

whether 는 '' 를 지정 선언(spec) 한다.
반드시 you have any 이하에 대한 '실지-여부' 를 알고 싶다는 것이다.

그런데 화자는
그 trouble 이 어떤 것인지 전혀 모르는 상태의 표현이다.

만약 화자가 그 trouble 의 내용이 어떤 것인지를 안다면
다음 ① 처럼 some 이 쓰였을 것이고

'실지-여부' 를 구문에 선언해주는 whether 가 아니라
that 으로 '' 짓는 선언을 했을 것이다.




①  You should like to know that she has  some trouble at home .
  ( 그녀 집에 문제가 있다는 것을 알아야 한다 )


whether 와 that 보문소에 따라
any 와 some 이 선택되는 현상으로부터 유추해 볼 수 있는 것은

some 이
'' 을 가지고 있는데 반해서

any 는 '어느 것도 단정하지 않는 비단정의 특성' 을 가지고
서로 상반된 대상을 선택 지정한다는 점이다.




②  Any  one of you will do . ( 아무나 한사람 해주면 돼 )
③  Some  one of you will do . ( 누군가 한사람 해주면 돼 )



any 의 수식 대상은 one 이지만
of you 에 의해 의미 보충이 주어져 있기 때문에

any 는 you 에 해당하는 하나하나 모두를 총칭적으로 전제하고
그들 가운데 하나를 one 으로 발화한 것이다.

다시 풀어보면 '너희들 중 어느 누구라도' 라는 말을 한 것이며
누구라고 구별하지 않는 '비단정의 하나하나 전부' 가 one 에 해당하게 된다.

이에 반에
③ 구문은 some 이 one 을 수식했기 때문에

one 은 you 를 전제로 하는
'나뉘어 구분되어있는 어떤 특정의 사람' 으로 단정 지어진 한 사람이 되는 셈이다.

두 비교 구문에서 드러난 문법적 작용을 보면
두 어휘는 서로 상반된 역할기능이 있음을 유추할 수 있게 한다.




④   some   =   단정
⑤       any    =   비단정



참고로
다음 ⑥ 의 anyone 은 ② 구문의 any one 과는 다른 문법 적용을 받는다.




⑥  Anyone  will do .
  ( 누구든지 간에 하려든다 )


⑥ 에서의 anyone 은 '어떤 자이든지', '누구든지 간에' 라는 의미이기 때문에
특정 범위가 없는 '전체' 이게 된다.

그렇지만
any 의 비단정적, 비구별의 기능적 의미가 one 에 결합되어진 것으로 보아야 하므로

any + one 은
'비구별의 하나하나' 로 나뉜 전체라는 의미로 자질화된 것으로 보아야 한다.

다시 말해, 이런 사람이든지 저런 사람이든지
'어떤 사람이든지 간에' 라는 의미로의 대상 지칭이 anyone 이다.

이것은 any 가
'따로 구별되어지는 것 하나 하나 모두' 를 다루는 특성이 있는 것으로 간주하게 해준다.
5. I will do anything but that.
( 그것 말고는 뭐든 하겠다 )

anything 을 '뭐든지' 라고 해석하는 것은
'비단정의 하나하나 전부' 를 가장 잘 설명해준다.

그리고, but 이 '앞에서 말한 내용에 따르면
안그래야 하는데 의외다' 라는 작용개념으로 '' 을 선언했으므로

이 구문을 그렇게 풀어본다면 다음과 같다.


① ( 어떤 것이라도 하겠지만 that 이 지칭하는 것만은 의외로 그렇지 않다 )


이 본문 5. 는 any 가
비단정의 하나하나 전부, 즉 '비-구분' 의 대상 모두를 말하는 긍정문이지만

but 을 선택해서
that 만은 그렇지 않다는, not 없는 부정을 한다.

이것은
any 의 특성인 비단정이, 부정의 여지를 갖게 했다고 볼 수 있다.

만약 '단정' 의 some 이 쓰인다면
그 대상이 구체적이게 되므로 부정의 여지가 남아있지 않을 것이다.

다음 ② 표현은 그와 같은 부정의 여지가 잘 적용된 글이다.




②  I doubt that  anyone did it .
  ( 나는 누군가가 그것을 했다고 생각지 않는다 )


② 의 해석을 달리 풀어보면
'아무도 그것을 했다는 생각이 들지 않는다' 라는 말이고

이런 해석은
doubt 동사의 의미적 부정이 any 의 비단정 특성에 부합했기 때문이다.
6. I'd like wine, if you have any.
( 네가 갖고 있는 것이 있다면, 난 포도주로 하겠어 )

포도주를 마시겠다는 의사 표시를 하면서
you 에게 있을 wine 의 '존재 여부' 에 따라 결정할 것이라는 말투다.

당연히 화자 I 는 you 가 가지고 있는 포도주의 종류나 양을
모르는 것이여야 하고, 또 any 로 그렇게 표현했다.

여기서도 any 는 '아무 종류의 포도주든' 혹은
'어느 정도의 양이든' 이라는 비구별로 표현했을 것이다.

다음 ① 구문도 같은 의미 표현이다.




①  There is little water, if  any .
  ( 물이 거의 없다 )


little 의 역할기능에 따른 작용개념은 '' 이기 때문에
① 의 if 이하 any 는 '있다 치더라도' 정도의 표현이다.

이것에 따라 우리말로 옮기면 심층적 의미로는 이렇게 된다.


② (혹 물이 있다 쳐도, 없는 거나 다름없이 적다)


지금까지 살펴본 any 가 쓰인 구문에서의 특성을 정리하면 다음과 같다.



첫째, any 는 '구분되어지는 것의 지정'' 이다.

둘째, any 는 '비-긍정' 의 요소가 아니라 '비-단정' 의 요소이다.

셋째, any 는 '심층에 전제된 전체' 로부터 나누어지는 것' 하나하나 전부' 이다.
