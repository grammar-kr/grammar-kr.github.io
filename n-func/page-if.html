접속어 if 는
실지-여부를 지정하는 whether 와 유사성을 보이는 역할기능으로 작용하는데

이 접속어는
가정을 나타내는 대표적 기능어로 문법화 되어 있다.

if 가 발화되는 통사적 위치에서 유사성을 보이는 whether 와는
그 나름의 변별되는 역할기능과 작용개념이 있기 때문일 것이다.




①  Do you know          if  the shops are open ?
  ( 가계들이 문을 열었을까? )
②  Do you know  whether  the shops are open ?
  ( 가계들이 문을 열고 있든? )


이렇게 두 단어를 선택적으로 사용해서 말하면
어떻게 인지해야 하는 것일까.

해석된 의미에 따라보면, ① 에서는 만일의 상황을
② 에서는 실제의 상황을 말하고 있다고 변별해야 할 것이다.

이것만이 아니다.
if 는(whether 처럼) '..인지, 아닌지' 라는 상반성이 심층적으로 전제되어져 있는 듯한데

or not 이 수반될 때와 그렇지 않을 때, 즉 '그렇고' 와 '그렇지 않고' 의 여부는
어느 범위까지를 적용해주어야 하는지도 설명되어야 한다.

또
whether 와는 다른 특성을 보이기도 있는데

whether 와는 달리
if 절 자체를 부정문으로 표현해도 된다는 점이다.

그뿐만 아니라, if 뒤에 to-부정사가 연이어질 수 없다거나
전치사의 목적어가 될 수 없다는 것 등이 문법 현상으로 존재하므로

다음에 제시되는 구문들의 분석을 통해서
이 어휘의 역할기능과 그에 따른 작용개념을 확인해보자.
1. We may boycott the summit if he is hanged.
( 그가 처형될 경우 우린 정상회담을 거부할 수도 있다 )

이 본문 1. 은 가장 전형적인 가정의 구문 형태로,
이 표현에서 boycott 이라는 동작은 발생하지 않았다.

왜냐하면 이 구문의 if 이하는
아직 실행되지 않은 일을 가상으로 표현한 '조건' 이기 때문이다.

접속어 if 를 '가정' 의 대표적 표현으로 이해하고
우리말 '...면' 에 해당하는 개념으로 내재화 하게 되지만

그러나
다음 구문 ① 을 보면 다르다는 것을 느끼게 된다.




①  Ask her  if he will come .
( 혹시 그녀가 올려는 지 물어봐라 )


해석된 우리말의 의미에 따라 보면
if 는 우리말 '...면' 에 해당하지 않을 뿐 아니라, '가정' 이라기보다 '예측' 에 가깝다.

또 이런 경우에는 '...지' 라는 우리말 연결어미를 떠올리게 되어
if 의 기능을 나누어 인지하게 되는데

다음 구문을 보면
둘 모두를 적용해봐도 부족함이 느껴질 것이다.
2. I don't know if it is true.
( 그게 사실인지를 난 모른다 )

본문 2. 를 다음과 같이 두 표현 형태로 비교해보자.




①  I don't know  whether  it is true .    
                ( 나는 그것이 (실지로) 사실인지를 모르겠다 )
②  I don't know          if  it is true .    
                ( 나는 그것이    (혹시) 사실인지를 모르겠다 )


① 과 ② 의 의미 차이는 whether 와 if 가 만들어냈고
두 어휘는 통사적으로 같은 위치에 있다.

① ② 의 구문이 표현하는 의미가 서로 같을 수 없다는 것은 곧,
서로 다른 두 어휘의 변별을 말해주는 것이므로

각각의 의미를 다음과 같이 다르게 부여해야만 한다.


① 은 실지로 true 인지(아닌지)
② 는 어쩌면 true 인지(아닌지)


우리말 조사 '...지' 는 서로 상반되는 사실을 대조시키는 연결어미인데
이런 해석이 자연스럽고도 의미 전달에 있어 충분한 것으로 볼 때

if 는 분명히 '..인지 아닌지' 라는
상반되는 대조의 성향을 어휘의 기본역할로 가지고 있는 듯하다.

그렇다면 if 는 대조되는 둘의 의미를 표층에서도 드러내고 있어야 하는데
해석인즉 그렇지 않다.

우리말 해석에서는 'true 인지-아닌지' 라고는 하지 않는다.
단지 앞의 'true 인지' 만이 표층에 드러날 뿐이다.

그러나
대조적 요소가 우리언어 개념상으로 있어야 한다면

'true 인지-(아닌지)' 가 아니라 'true 인지-(어쩐지)' 정도의 미정 요소를
대조의 짝으로 부여할 수는 있을 것이다.




③  I don't know  if  that's such a good idea .
  ( 그게 그렇게 좋은 생각일련 지(어떨련 지)를 모르겠다 )



3. He asked me if Jenney could speak Spanish.
( 그는 내게 제니가 스페인어를 말할 수 있는지를 물었다 )

앞에서 말한 것에 따르면
이 본문 3. 에서 if 가 드러내고 있는 대조는 무엇인가.

본문 1. 에서는, if 가
'실행되지 않은 동작 상황을 가상으로 표현하는 조건문' 을 지정해주는 것으로 간주했었다.

그렇게 간주하고 보면
if 이하는 실지로 행하고 있거나, 행했던 동작이 아니다.

could 라는 '가능성' 의 단어로
주어의 동작을 한 것도 그것을 말해주고 있다.

if 가
추정적으로 지정하는 'true 인지(그렇지 않은지)' 는

Jenney 가 spanish 를 말할 수 있는지(그렇지 않은지) 로써
그것이 심층에서 상반되고 있다고 볼 수 있다.

여기서 보여주는 상반성을 '여-부' 라고 규정해보자.
참고로 '여부' 의 우리말 사전은 '그러함과 그러지 아니함' 으로 되어 있다.

그러면 '..인지' 의 표층 요소와 '..아닌지' 의 심층 요소는
if 절 주어의 동작 행위로 상반되는 것일까.

해석을 보면 다음 ① 구문은
'인지-아닌지' 로는 바꿀 수 없게 됨을 알 수 있다.




①  I'll be surprised  if  he doesn't win .
  ( 난 그가 우승하지 못하면 놀랄게다 )


① 구문에서 if 는
'만일 그가 승리하지 못하면' 이라는 가정 상황 앞에 접속되어 있다.

만약 승리하지 못하면 놀라게 될 것이라는 표현이지만
심층에서는 '거의 이긴다' 라는 전제하에 표현할 수 있는 어법이다.

그렇다면, if 가 드러내주는 상반성은
동작을 '하고-못하고' 가 아니라 '혹시-여부' 를 상반시키고 있는 셈이다.

다시 말해서 win 을 못할 가능성은 표층에 표현했지만
win 할 가능성은 심층에 두고 의미를 상반시키고 있다는 말이다.


② 혹 그러지 못함(doesn't)    +    거의 그러함(does)
        (표층적 표현)                    (심층적 전제)


① 의 구문이 doesn't 과 does 가 상반되고 있듯이
본문도 if 에 의해서 '여-부' 가 상반될 때

could 로 표현된 '가능성' 이 상반되고 있으므로
상반되는 여-부의 대상은 동작이 아닌, '가능성' 이 된다.
4. I don't care if your car breaks down or not.
( 네 차가 고장이 나든 말든 개의치 않는다 )

본문 4. 는 if 종속절에 or not 이 수반되어 있는데
or 접속어 전,후에는 의미적으로 상반되는 표현이 주어질 수밖에 없다.

왜냐하면 접속어 or 가
'' 이라는 의미로 작용하기 때문이다.

그렇지만, 이미 살펴본 여부 상반과
이 본문에서와 같은 or 접속이 주는 의미 상반은

A 의 여부와 B 의 여부, 즉
양쪽의 '여-부' 가 or 을 두고 대조를 보여주게 된다.




①  I don't care  if  your car breaks down or not .               
                 ( 신경 쓰지 않는다. 고장이 나든지(안 나든 지)   (아니면) 그 반대의 경우든)


그러면
not 이 무엇을 부정하고 있느냐를 확인해보자.




②  I don't care  if  your car breaks down or not .     
                                                              ( 고장이 나든 지 )   or ( 안 나든 지 )


언 듯 이렇게 or 을 두고 양쪽을 '여 와 부' 로 이해해도 되는 듯하지만
③ 에서처럼 or not 부분을 복원해보면 모순이 나타나게 된다.




③  I don't care  if  your car breaks down  or  if  your car does  not  break down .
                                                  ( a )                             or                  ( b )


or 접속어 자체는
'여-부' 의 상반 기능이 없고, 단지 '그렇지 않으면' 이라는 대조 기능만 있다.

이것에 따르면 ( a ) 와 ( b ) 모두에서
'여' 는 표층에 드러내고, '부' 는 심층에 작용하게 하는 기능이 if 의 역할이라고 할 만하다.

이것은
( a ) 에서 만일의 경우의 동작, 즉 혹 그럴 가능성의 상황이 표층으로 드러날 때

심층에서는 그럴 가능성이 배재 된다는 말이다.
다시 말해 '네 차는 고장 날 가능성이 거의 없음'이 마음에 작용하고 있다는 것이다.

그렇지만, 구문 ( a ) 에서는
만일의 경우에 행해질 동작, 즉 혹 그럴 가능성의 상황이 정반대가 된다.

문제는 ② 구문에서의 or 부가절에 홀로 쓰인 not 을
③ 의 구문처럼 or 부가절에서 if 와 S + V 이하가, not 만 남기고 생략된 결과로 볼 경우

( a ) 와 ( b ) 에서 표현하는 의미가
순서만 역순이 되었을 뿐, 같은 말을 반복한 꼴이 되고 만다는 점이다.


④ 혹 고장이 나든(안 나든), 그렇지 않으면 혹 고장이 안 나든(나든)


따라서
if A or not B 라는 종속절을 if A or not if B 까지 확대시켜야 한다.




⑤  I don't care  it is   if  your car breaks down  or  it is   not  if  your car breaks down



이렇게 복원해 보면
부가절의 not 은 동작을 부정하는 것이 아니라

if 의 혹시 여부를 부정하고 있음이 드러나
앞 3. 에서 언급한 '혹시-여부' 의 상반을 뒷받침해준다.

이제는 '만일' 즉
혹 그럴 가능성의 경우를 적용해서 심층적으로 해석해보자.


⑥ '나는 네 차가 혹 고장이 나든(안 나든) 아님 거의 고장이 나든(안 나든) 개의치 않는다’
                         if(혹시나)                    or     not if(역시나)


⑤ 의 구조에 따르면 or 이하의 부가절에서 not 은 if 를 부정해서
or 를 두고 양쪽에 '혹' 그럴 가능성과 '거의' 그럴 가능성을 대조시키고 있는 꼴이 되는데

또다른 근거 구문을 보자.
5. If it rains or not doesn't concern me.
( 비가 오든 말든 관심 밖이다 )

이 장에서 확인하고자 하는 것은, 본문 5. 에서와는 달리
if 뒤에 곧바로 or not 이 발화될 경우이면 왜 비문이 되느냐는 것이다.




①  If  it rains or not  doesn't concern me .
②*  If or not  it rains  doesn't concern me.



이 비문 ② 의 주어절을
앞의 제시 본문 4 - ② ③ 처럼 이해하고 복원시켜 보자.




③  If  it rains      or    If  it does  not  rain   doesn't concern me .
( 혹 비가 올 경우 )  or     ( 혹 비가 오지 안 올 경우 )


언뜻 우리의 언어개념상으로는
의미적으로 문제가 되지 않는 문장처럼 느껴져서 옳은 복원처럼 보일 수 있다.

그렇다면
① 에서 it rains 가 중복되어 부가절에서는 생략된 것과 달리

② 에서는 뒤가 아닌 앞에서 it rains 를 생략시키고
의미 전달의 최소 요소만을 남겨둔 꼴인데 비문이라는 점이다.

이것은
③ 의 복원 구문이 ② 의 심층 구문이 아니라는 것을 말해주는 셈이다.

그래서 이번에는 앞의 4 - ④ 에서 도출해 놓은
'혹 가능성' or '거의 가능성' 을 받아들이고 설명해 보자.




④  If  it rains or not  if  it rains  doesn't concern me .



④ 는 not 이 if 를 부정했으므로
'혹 가능성 or 거의 가능성' 의 구조를 갖는다.

그런데 or 자체는 '여-부' 의 상반기능이 없고
단지 '그렇지 않으면' 이라는 대조 기능만이 있으므로

or 로의 '그렇지 않으면' 이라는 접속선언은
바로 앞 상황과 상반되는 대조적 상황임을 말하려는 것이 된다.

그래서 바로 앞 상황과의 관계로 or 접속비교를 생각하면
다음 ⑤ ⑥ 처럼 결속되어 작용하는 것으로 분석할 수 있다.




⑤  (If it rains)  or  ( not If  it rains)  doesn't concern me
⑥  (If) or  ( not If) it rains  doesn't concern me



두 구문을 어휘나열에서 살펴보면
or 에 의해서 상반-대조되는 대상이 다르다.

⑤ 구조에서는
If it rains or (not if it rains) 이므로

if 가 or 의 양쪽에서 보문소로 작용하며
it rains(-비내림 상태)를 이끌면서 or 대조를 이룬다.

그러나 ⑥ 은
(If) 와 (not if) 가 or 대조의 직접 대상이라서

상반 대조되는 두 어구 자체가, 또 하나의 새로운 보문소처럼
'혹시(if) 가능성' 과 '거의(not if) 가능성' 이라는 한 의미로

it rains 의 문장 성격을 선언하는 꼴이 되기 때문에
비문이 분명하다.

그리고 ④ 의 구문에서 보면, or 에 의한 대조는 양쪽에서
보문소인 if 가 상반되어져 혹(if) 그럴 가능성과 거의(not if) 그럴 가능성이 대조는 되지만

'혹' 과 '거의' 는 확률의 차이일 뿐이라서
'인지-아닌지' 와 같은 반대적 요소를 내포하지 않는다.

따라서
⑥ 과 같은 표현이 비문인 것은 If 와 not 의 or 결속 때문이며

if 의 의미는
or not 에 의해 상반적으로 대조될 수 없는 자질로 되어 있다고 이해할 수 있다.
6* He wondered if to accept the offer or not.
( 비문 -- 그는 그 제안을 받아들일지 말지를... )

본문 6. 도 비문이다.

if 뒤에 곧바로 이어지는 to + verb(원형) 구조로의 의미 표현이
영어에서는 성립되지 않는 듯하다.

그렇다면 to + verb 라는 특성이
if 의 대상이 되지 못하는 현상을 어떻게 이해해야 하는가.




①* He wondered  if to accept  the offer or not .



① 구문에서 if 이하의 'to + 원형동사' 를 지배하는 의미상의 주어는
주절 주어 he 가 되어야 한다.

생략은 복원 가능성 위에서 행해지게 되기 때문에
여기에서는 he 가 아니면 있을 수 없다.




②? He wondered  if he would  accept  the offer or not .



앞에서 살펴본 바에 따르면
if 는 '가능성 여부' 즉 '혹 그럴 경우 여부' 를 지정해 주었다.

그래서 '혹시 여부' 를 이 구문에 적용해보면, ② 표현은
자기가 그 제안을 받아들일 건지 말건지를 의아해하는 이상한 표현이 되고 만다.

그럴지라도
구문 형태상, 복원된 ② 구문까지를 비문이라고는 할 수 없을 것이다.

② 처럼
복원된 구문으로부터 He would 가 생략(?)되고

to 로 대치되어 표현되었다고 했을 경우가 ① 과 같은 비문이므로
이와 같은 생략과 복원은 생각할 수 없다.

따라서 to 를 He would 의 대용어쯤으로 보고 비문 현상을 이해하려 들면 안되고
to 가 가지고 있는 '당위성' 으로 구문 ① 의 비문 현상을 이해해야 한다.




③      if           to      accept .
    (혹시 + 그래야 하는 것)   (이것)



if 에 적용시키는 '..지', 혹은 '..면' 이라는 우리말 번역 요소는
화자가 주어의 가능성을 '예측' 하는 데서 비롯된 말이지, 결코 '동작' 을 말하는 것이 아니다.

그런데 if + to(verb) 형 표현은
주어의 행위 가능성, 즉 if 의 상황이 당위적일 수밖에 없다고 말하는 어휘 선택인 셈이므로

결국 if + to(verb) 의 나열은 '만일의 경우(if)' 라는 희박한 가능성을 말하고 나서
그 가능성에 '' 라는 당위성을 부여한 꼴이 되고 만다.

희박한 가능성은 '허상' 이라고 할 수 있는데

'그래야 할 당위적 대상'을 두고 작용하는 to 앞에는
허상을 말하는 if 선언이 올 수 없을 것은 자명할 것이므로

if 는 to 에 의한 당위성을
'혹시 여부' 로 선언해주지는 못한다고 이해할 수 있다.
7* It depends on if he will support us or not.
( 비문-- 그것은 그가 우리를 도울건지 말건지에 달려있다 )

7. 같은 표현은
if 앞에 전치사 on 이 결합되어 있어서 비문으로 알려지는 형태의 구문인데

이 구문을
복합동사(구동사)라는 문법 틀에서 살펴보면 모순이 있다.

'depend on' 을 하나의 동사처럼 보는 문법이론에서는
on 을 부사로 간주한다.




①  depend + on .
    ( 동사 )      ( 부사 )


그런데
on + if 가 쓰여 있는 7. 표현을 비문이라고 말한다는 것은

on 이 부사적 기능을 하는 것이 아니라
전치사 기능으로 if 에 작용하고 있음을 반증하는 셈이다.

의미적으로 분석해도 on 은
분명히 if 를 전치하고 있기 때문에 비문이 되는 것임을 알 수 있다.

전치사 on 은, 전치한 대상 앞에서
'' 의 상태에 있음을 밝히는 '상태사' 로 작용하게 되는데

그렇게 밀착된 대상이 허상인 if(혹시) 상황이라고 표현해주는 것이 되어
의미적으로 성립할 수 없을 것이다.

이처럼
이 본문에서의 if 가 on 의 밀착 대상이 될 수 없는 것으로 볼 때

여기에서의 if 는
없는 것을 가정해서 '혹 그럴 경우' 라고 의미-지정을 했음에 개연성이 더해진다.
8. I'm not sure if he is here yet.
( 나는 혹 그가 아직 여기에 있는지를 잘 모른다 )

형용사에 이어져 표현됨으로 해서
형용사의 목적어라고도 분석되는 본문 8. 에서의 if 명사절을

명사절과, 또 명사절과도 비교해 보자.




①  I'm not sure  (   )  he is here yet .
  ( 그가 아직 여기에 있다는 것을 확신할 수 없다 )
②  I'm not sure   if  he is here yet .
  ( 그가 아직 여기에 있을련지를 확신할 수 없다 )


두 보문소 if 와 (no_spec) 이 선언하는 의미에 충실해 보면
①, ② 와 같은 해석을 낳는다.

또 보문소 whether 와 if 에서의 차이는
'' 와 '혹시-여부' 가 변별요소로 작용하고 있을 것이다.




③  I'm not sure  whether  he is here yet .
  ( 그가 아직 여기에 있는지의 '실지 여부' 를 확신할 수 없다 )
④ I'm not sure         if  he is here yet .
  ( 그가 아직 여기에 있는지의 '혹시 여부' 를 확신할 수 없다 )




9. If you had done as I told you, you would have succeeded.
( 내가 시킨 대로 했더라면 너는 성공했을 거다 )

9. 같은 구문은, if 가 드러내는 여-부 중에서
표층에서는 '여', 즉 '혹 그랬을 경우' 만 나타나지만

그 의미 속에서는
반대 요소인 '부' 가 강하게 작용하는 표현이 되고 있다.

다시 말해
'거의 그러지 않았다' 라는 저의를 가지고 한 말이 되는데

이 같은 표현을 '현실과 반대되는 가정' 표현으로 분석하는 것도
반대 요소인 '부' 가 작용하고 있음을 인정하는 것이 된다.




①  If you had done as I told you, . . .



① 의 구문은 '과거완료' 형으로 표현되어 있는데
'혹 과거에 그랬을 경우' 라는 말이 자연스럽게 적용되어 진다.

이것은 if 의 혹시 여부(가능성 여부) 라는 역할기능을
과거의 상황에서 적용시키는 차이 뿐이다.

도출해 본 if 접속사의 역할기능과 그에 따른 작용개념을 정리하면 이렇게 된다.



첫째, if 는 여-부, 즉 그러함' 과 '그러지 아니함' 지정한다.

둘째, if 의 '여-부' 는 if 절 안에서 '혹시 상황' 으로의 상반이다.

셋째, if 의 '여부 지정' 은 만일의 상황, 즉 '혹 그럴 가능성의 경우임'을 말한다.


역할기능 : 혹시-여부
작용개념 : 혹 그럴 가능성의 경우
