so [ 정도-규정 ]                       보통의 그러함과는 달리         유튜브-영상 : shots

글자크기

접속-부사로 기능하는 so 를 대하면
먼저 떠오르는 우리 단어는 '그래서' 일 것이다.

그래서 라는 우리말에 대한 사전적 규정은
'앞의 사실이 뒤의 사실의 원인, 근거, 조건이 됨을 나타내는 접속부사' 라고 되어 있는데

이러한 우리말 의미는
so 가 접속사로 쓰이는 구문에서 아주 적절한 우리말 번역어가 되어 준다.




①  I am tired,  so (that) I cannot go on.
  ( 나는 지쳐서 계속할 수가 없다 )


① 구문의 so 는
tired 상태를 다시 언급하는 '그래서(-피곤해서)' 가 되어 주지만

다음의 구문처럼
so 가 접속부사가 아닌, 수식 부사로 쓰이면 다르게 나타난다.




②  I am so tired  that I cannot go on.
  ( 나는 계속할 수가 없이 몹시 지쳤다 )


② 에서의 so 는
tired 된 상태를 다시 언급하는 것이 아니라

tired 되는 정도를 so(-너무) 로 말하고
그 '너무' 상태에 따른 결과를 that 이하로 구체화한다.




③  I am so tired  as I cannot go on.
  ( 나는 계속할 수 없을 만큼 치쳤다 )


③ 구문에서는
so 에 의해 수식되는 tired 된 정도가 as 이하로 여길 만큼이다.

세 구문 모두에서
'그래서, 너무, 그만큼' 등은 해석의 편의상 선택되는 어휘일 뿐이므로

so 가 쓰인 구문을 분석하며
so 의 역할기능과 그에 따른 작용개념을 도출해보자.
1. He asked for a pen, so I gave him one.
( 그가 펜을 요구해서 하나 줬다 )

본문 1. 은 so 가 접속부사로 쓰인 구문 형태로
우리말 '그래서' 가 잘 적용된다.

그가 펜을 요구했고
그 요구에 따라 그에게 펜 하나를 주었다는 표현인데

이것을 역으로 생각하면
그가 요구하지 않았으면 주지 않는 상황에서 출발한다.

다음 구문들에서 그 같은 역의 상황들을 알아보자.




① He is ill,  so we must call on him .                                  
  ( 그가 아파서 우리는 방문해야 한다 )-- 아프지 않으면 지금의 방문은 없을 것이다 --
②  If he is going, then  so am I .                                  
  ( 그가 간다면 그때는 나도 간다 )-- 그가 가려하지 않으면 나도 갈 리 없다 --
③  I was excited,  so I couldn't get to sleep .          
  ( 나는 설레서 잠을 이룰 수가 없었다 )-- 설렘이 없었다면 잠을 잘 수 있다 --


제시된 ① ② ③ 구문들을 살펴보면
so 를 두고 앞뒤에 표현된 상황들이 인과 관계를 가진다.

앞의 상황이 있기 때문에
뒤의 상황이 결과로 발생한 것이다.

그러나
이와 같은 원인과 결과 사이에 개입되어 작용하는 so 가

표면상으로는
원인이 되는 상황을 전제로 제시하며 접속하는 기능을 하지만

그렇게 제시된 상황은
반대되는 상황을 심층적 배경에 두고 형성된 것일 수밖에 없다.
2. We arrived early, so (that) we got good seats.
( 우린 일찍 도착해서 좋은 자리를 잡았다 )

앞 주절에서는 일찍 도착한 상황을 말해주고 있고
뒤 종속절에서는 일찍 도착한 상황이 빚어낸 결과를 표현했다.

서로가 결속되어 하나의 표현을 만든 두 구절이
'원인-상황' 과 '결과-상황' 을 표현했다고 볼 때

so 는 그 둘 사이에서 작용하는 관계사(접속사)인 셈이다.

여기서의 '원인-상황' 은 남달리 일찍 도착한 상태일 것이며
그것이 원인이 되어 좋은 좌석을 잡았다는 의미 표현이므로

so 이하의 that 절에는
일찍 도착했었기 때문에 빚어진 결과가 종속되어질 수밖에 없고

그래서 '그렇지 않은 상황' 즉

남들과 같은 시간에 도착해서 빗어질 반대상황과
심층적으로 비교되어있는 표현임을 유추할 수 있다.




① I was excited,  so I couldn't get to sleep .
  ( 나는 설레서 잠을 이룰 수가 없었다 )


① 의 표현에서도
so 앞에 나열된 원인-상황은 '흥에 겨운 상태' 이고

이것이 계기가 되어 잠을 이룰 수가 없었다는 결과-상황이
so 이하에 서술되어 있다.

그런데 so 는
앞의 '흥에 겨운 상태' 를 다시 언급한 것이겠지만

뒤에 종속되는 결과-구문과의 관계에서 보면
so 는 '결과 상황' 앞에서 '원인 상황' 의 어떠함을 전제적으로 제시하고 있다.

그리고, 본문 2. 와는 달리 ① 에서는 (that) 도 없는데
그렇다면 that 삽입 여부는 어떤 차이를 내는 것일까.

that 이 드러내는 역할기능은 명시적인 '' 이다.
이에 반해 that 이 없는 no-spec 구문은, that-spec 구문과의 차이에서 볼 때

종속절에서의 표현을 '' 만으로 규정한 것이므로
두 경우를 변별해 적용해보면 이렇게 된다.

종속절 앞에서 so 를 선택해서 발화한 것은
있었던 어떤 한 상황의 '어떠한 정도' 를 심층적으로 규정지어 전제하고

이어서 접속시켜 표현해 줄 종속절의 성격을 규정할 때
that 으로 선언함으로써

앞 상황이 빚어낸 결과를 '단정' 지어 표현하느냐, 아니면
(no-spec) 으로 빚어진 결과를 '설정' 만으로 표현하느냐에 따른 의도상의 선택으로 볼 수 있다.

다음은 so 와 that 의 결합 구문을 살펴보자.
3. It was so cold that he was shivering.
( 날씨가 너무 추워 떨고 있었다 )

본문 3. 에서는
앞 2. 의 본문과는 달리, 콤마로 분리하지 않은 하나의 구문 안에서

그것도
so 와 that 이 분리되어 so 가 형용사 앞으로 나와 있다.

2. 의 본문에서는
so 가 해당된 원인 상황을 심층적으로 '규정지어서 전제' 시킨다고 했는데

그렇게 규정지어져 전제됨을 받아들여 보면, 본문 3. 의 경우에는
표층적으로 제시되는 원인 상황이 '유달리 춥다' 라고 규정되는 것으로 볼 수 있다.

왜냐면 그 배경 상황(그렇지 않은 상황)이 It was cold, 즉
so 로 수식되지 않은 만큼의 추위이기 때문이다.

그리고, that 이하로는
'그 상황이 만들어 준 결과를 구체화' 하는 보충 서술을 하고 있는데

He was shivering(그가 떨고 있었다) 이라는 사실이
that 으로 단정되었다.

이것들을 합하면 '그가 떨고 있을 정도로 추웠다' 라는 표현이 되어
다음 구문처럼 so 의 수식이 없는 형태와 뚜렷한 의미 차이가 있게 된다.




① It was cold  that he was shivering .
  ( 그가 떨고 있는 것은 추워서였다 )


so 가 없는 ① 의 구문은 It ~ that ~ 형 구문이고
'그가 떨고 있는 것은 추워서였다' 정도로 의미가 약화된 것으로 볼 때

so 가 추위의 정도를 강조했음이 분명해진다.

따라서 본문은
'보통의 추위와는 달리 떨고 있을 정도' 라는 원인 상황이 so 에 의해서 전제가 되고

that 으로는
보통 추위와는 달리 나타나는 반응을 단정지어서 구체화한 구문 형태임이 드러난다.

그렇다면
화자가 so 를 말할 때는 이미 '보통의 추위 때와는 달리' 라는 의미로

원인 상황을 발생시킨 그런 정도의 추위라고 규정지어 강조하려는 의도에서
so 를 앞세워 발화하는 전제적 수식을 한 셈이다.




② She was  so happy that tears ran down .
  ( 눈물이 흐를 정도로 행복했다 )


so 에 의해 행복 상태가 강조된 원인 상황은
'보통 때 느끼는 행복과 달리' 라고 제시되었고

그 행복의 강도는
that 이하에서 '눈물이 흘러내릴 정도' 라고 서술되어 있는데

역으로, 보통 때 행복을 느끼는 정도가 눈물이 흐를 정도는 아니다로
심층에 있음을 유추할 수 있다.

다음의 구문을 보며 확인해보자.
4. Switch the light off so that we can sleep well.
( 잠이 잘 들게 불을 꺼라 )

4. 의 본문은
so 의 수식으로 강조되는 원인-상황이 '불이 켜져 있을 때와는 달리' 가 된다.

'불을 끄라' 라는 말은 불이 켜져 있음을 전제로 하는 표현이고
배경이 되는 반대의 상황은 불이 켜져 있는 상태일 것이므로

표층적으로 드러나야 하는 실제상황은
그러함과는 달리, 즉 보통 때의 그러함(불이 켜져 있을 때)과는 달리 라는 결과가 된다.

이런 결과는 '보통의 그러함과는 달리' 라는 우리말이
심층적 작용개념으로 개연성이 있음을 말해준다.

다음 구문에서 이 개념을 적용해 보자.




① I called him again  so that he noticed me .
  ( 그를 다시 불러서 나를 알아보게 했다 )


again 의 발화는
이전에 call 이 '한 번 있었음'을 암시한다.

그렇다면
이 구문에서의 표층적 결과-상황은 called him again 이고

배경적 상황은 call him 이 되어
결국 다시 부른 결과로 처음과는 달리 분명히 그가 날 알아봤다는 말이 된다.




② Please articulate clearly  so that everyone can hear you .
  ( 모두가 알아들을 수 있게 또박또박 발음해다오 )


구문 ② 에서도 또박또박 발음한 결과로
이전 발음의 정도와는 달리, 모두가 알아듣게 말하라는 표현이다.
5. He is not so clever as he looks.
( 그는 보기와는 달리 그렇게 똑똑하지 않다 )

so 가 가지고 있는 또 하나의 중요한 기능은 '비교성' 이다.

영어의 문법을 배울 때
as ~ as ~ 구문과 so ~ as ~ 구문을 비교해가며 배우게 되는데

이것은
두 표현 형태를 서로 변별되게 인지해둬야 하기 때문일 것이다.

앞에서의 구문에서도 원인 상황을 말할 때는
'그렇지 않은 보통의 그러함'과의 비교에서 발생했었다.

본문 5. 의 not so clever as ~ 에서 so 가 제시하는 원인 상황은

앞 구문들과는 다르게, 결과적 상황이 부가되던 종속절 위치에
비교로 인용된 as he looks 이 있다.

이 표현을 분석해보면, not 에 의해 so 이하가 부정되므로
'보통의 그러함과는 달리' 가 '보통의 그러함 처럼' 으로 의미가 변화되어

as he looks, 즉
그가 평소 비춰진 것처럼 그렇게 영리하지 않다는 표현이 된 것이다.

그래서
not so ~ as ~ 표현형에서의 so 에도 이 개념을 적용하면

as 이하 모두가 so 수식의 전제 자료가 되어서
다음 ① 에서처럼 보통의 그러함과는 다르지(not) 않다는 표현이 된다.




①  He is not  so clever as he looks .
  ( 그는 보기와는 달리 그렇게 똑똑하지 않다 )


so 의 앞뒤에
원인-상황과 결과-상황이 표현되던 것과는 다르게

① 의 표현은
so 가 수식한 clever 의 정도를 보통의 그러함과 다르지(not) 않다고 밝히고서

보통(평소)의 그러함으로 여겨지는 비교 상황(as 이하)을
결과적 상황표현 위치에 인용시키고 있다.

그렇다면
so 가 형용사의 상태를 비교 수식할 때의 비교기능을 생각해보자.




②  Tom  is not  so     old     as William .
  ( 탐은 윌리엄만큼 늙지 않았다 )
③  A cat is not  so faithful as dog .
  ( 고양이는 개와는 달리 충실하지 않다 )


② 와 ③ 구문들처럼, so 가 형용사의 상태를 비교할 때
so 의 비교기능은 심층적으로 다음과 같은 성향을 드러낸다.




④  He looked  so  funny  that I couldn't help laughing .
  ( 그는 너무 우스워 보여서 내가 웃지 않을 수 없었다 )
⑤  Nothing is  so precious as time .
  ( 어떤 것도 시간만큼 그렇게 소중하지 않다 )


④ 표현에서 so 의 비교기능은, 표층에 제시된 상황(그런 상황)을
심층적으로 작용하는 반대 상황(그렇지 않은 상황)과 대조시키는 '상반 비교' 인데

웃지 않을 수 없는 his funny look 과
안 웃어도 되는 his funny look 이 비교되고 있다.

이에 반해서 ⑤ 의 표현에서 so 에 의해 비교되는 것은
시간의 '소중함' 과 그 외 것들의 '덜 소중함' 이 상반적으로 비교되었다.

차이가 있다면, ④ 표현에서는 that 으로 결과-상황를 '단정'하고 있기 때문에
so 에 의한 비교의 결과가 that 이하에서 '명시' 되고

⑤ 에서는 so 에 의한 비교의 결과가 as 로 '간주(-여김)' 되기 때문에
비교될 상황이 as 이하에 그렇게 여겨지는 대상으로 인용되는 꼴이다.




⑥  Time is  as precious as anything else .
  ( 시간은 다른 모든 것들만큼이나 소중하다 )


⑥ 구문에서 as 가 해 보이는 비교기능은, 표층적으로 간주 되는 상황(시간의 소중함)과
심층적으로 그렇게 간주해 보는 상황(-그 밖의 모든 것들)과의 동등비교이다.

구문 형태적으로는
so 의 위치에 as 가 대치되어 있는 구문으로 볼 수 있어서

as 의 역할기능 '여김' 에 따르면

so 의 비교 수식에서 작용한다고 보는 '그렇지 않은 상황' 대신
'같은 상황'(-as 의 동등비교 상황)이 작용하는 셈이다.

달리 풀어보면

so 의 보통의 그러함과는 '달리' 가
as 의 보통의 그러함과도 '같이' 로 바뀐 꼴이다.
6. Don't run so fast as to become easily tired.
( 너무 빨리 달리지 마라 쉽게 지칠라 )

본문 6. 은
앞의 본문 5. 와 똑같은 so ~ as ~ 상반 비교 구문이다.

그런데, as 이하에서는 전치사 to 에 의한 to-부정사가 쓰였는데
as 로 인용해서 비교하는 대상이 명사구가 아니라 전치사구(to become)로 되어 있다.




①  Don't run  so precious as to become easily tired .
  ( 너무 빨리 달리지 마라 쉽게 지칠라 )


구문 ① 표현을 해석할 때
as to 를 '숙어화' 해서 해석의 편의를 가지려 하지만

그보다는
as + to V 로 이해하는 것이 더 문법적이다.

① 의 표현은
so 에 의해서 발생한 비교의 결과를 as 로 해서 인용하고

그 인용의 내용이
to + become 이라고 구체화해주는 형태의 구문이다.




②  Will you be   so kind as to lend me this book ?
  ( 너는 친절히도 내게 이 책을 빌려줄 수 있니? )


② 의 구문에서
so 에 의해 비교되는 kind 의 정도는 as 이하에 표현되는 '만큼' 이다.

그래서 as 를 구체화한 to lend, 즉
'빌려주려고 시도할 만큼' 이 so 에 의해서 강조되는 친절의 정도가 된다.




③  You must get up at 5 a.m.  so as to catch the first train .
  ( 첫차를 타려면 새벽 5시에는 일어나야 한다 )


③ 의 구문은
so 와 as to V 가 연속 나열된 형태인데

사전에서는 so as to V 를 숙어화해서
'V 하기 위하여' 라는 해석의 편의어를 제시하고 있다.

그러나
앞과 동일한 방법으로 이해하는 것이 더 생산적 문법 규정이 될 것이다.

so as (to V) 는 so that 구문과 문법적 기능이 다르지 않다.
다만 that 대신 as 가 발화되었고, as 이하에 전치사구를 수반시켰을 뿐이다.

따라서 so 에 의해서 제시되는 상반 상황을
as 이하에서 결과로 간주해 보는 비교 구문이 되고

이것을 발화순서대로 이해해보면 이렇게 이해된다.


'5시에 일어나야만 한다, 그러면(-그렇지 않았을 때와는 달리-) 첫 차를 잡을 수 있다고 여겨진다'




7. Be good and stay so.
( 그렇게 착하게 굴어라 )

7. 본문은 so 가 구문의 끝에 단독으로 쓰인 경우인데

여기서도 so 에 의해 전제로 제시되는 그런 상황(정도) 은
to be been good(착하게 굴기) 이다.

역으로
그렇지 않은 상황이 심층에 있음을 감안하면

화자의 말을 들은 당사자(청자)는
평소에는, 최소한 말하기 직전까지는 착하지 않았다는 말이 된다.

그리고, stay 하라고 말할 때
so 로 비교 수식한 만큼을 조건으로 제시한 것이므로

'보통의 그러함과는 달리' 라는 심층적 개념을 적용하면
so 의 역할이 선명해진다.




①  A man who should say  so must be mad .
  ( 그렇게 말하는 사람은 미쳤음이 틀림없다 )


① 구문에서, 그가 미쳤다는 표현을 한 것으로 느낄 수 있게 하는 어휘는
so 가 아니면 찾을 수 없다.

우리말로 '그렇게 말하는 사람은 미쳤다' 라고 하면
그가 어떻게 말했기에 미쳤다고 하느냐고 반문할 것이다.

그렇지만 so 의 작용개념을
'보통의 그러함과는 달리' 로 적용해보면 사뭇 다르다.

A man 은
a 의 수식으로 인해 특정인이 아닌, 된 일반인이다.

그리고
should 는 '' 라는 화자의 견해이다.

이것을 결합하면
so 가 지칭하는 상황은 다음과 같은 긴 심층적 해석을 낳는다.



보통 사람들과는 달리   그렇게 말하게끔 되어져 있는  이런 유형의 사람은 미쳤음이 틀림없다
            (so)                          (should say)                    (a man)



②  You don't say  so . ( 농담이겠지. -정말이야!- )


② 의 표현 구문은
문법적 어휘나열에서 분석하면 주어가 삽입된 명령문이나 다름없다.

그런데도 해석에서 보듯이, 이런 우리말 의역이 가능하려면
so 에 의한 상황(-정도)의 전제가 보통의 그것과는 다름으로 제시되었어야 할 것이다.


'너, 사실과(보통의 그러함과) 다르게 말하지 마라!
8. So sorry!
( 정말 미안해! )

'정말' 이라는 해석은 so 에서 비롯되었을 것이다.

그렇다면 단순히 sorry 라는 말로 표현했을 때보다
의미가 어느 정도나 강조되는 것일까.

물론 그 절대치는 말할 수는 없지만
상대적으로는 그 '정도' 를 가늠할 수는 있다.

so 가 비교 수식하는 '보통의 그러함과 달리' 를 적용해보면
'유난히 미안하다' 와 같은 말을 한 셈이라서 그렇다.




①  So Tom likes fishing, does he ?
  ( 그렇게도 탐이 낚시를 좋아한다며, 맞어? )


① 표현은
발화할 때 끝을 끌어 올려서 놀라움을 표현한 부가의문문인데

이 구문 서두에
표층적으로는 전제 상황이 없을지라도 심층에서는 제시되어 있다.

so 의 발화 의도를 수용하면
주어 Tom 은 보통의 그것과는 달라야하므로

탐이 다른 것(-취미)을 좋아하지 않는 것과는 달리
낚시는 정말(-정도)좋아한다는 표현을 했다면 '유달리' 라는 해석이 가능하고

만약 탐이, 다른 사람과는 달리 fishing 을 좋아한다는 것으로 표현했다면
'남달리' 라는 우리말 해석이 적절할 것이다.


지금까지 분석해 본 so 의 역할기능과
그에 따른 작용개념을 정리하면 이렇게 된다.



첫째, so 는 '있(었)던 상황(대상)' 을 전제로 하는 당면 시점에서의 '정도 규정' 선언어이다.

둘째, so 의 정도 제시는 이전의 '그렇지 않은 상황' 과 상반된 '그런 상황' 이다.

셋째, so 에 의한 '그런 상황' 은 보통의 그러함과는 다름으로 표현된다.

작용개념 : 보통의 그러함과는 달리
